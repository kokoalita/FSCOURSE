This file edscribes the steps to run and create react project 
to create a new project with vite
    npm create vite@latest
to install teh required libraries
    cd [dir]
    npm install
to run 
    npm run dev

    -- npm run dev --> Start the app locally for development
    -- npm run build --> Package the app for production
    -- npm run preview --> Preview the production build locally

to run json server 
    json-server --port 3001 db.json

to run the linter 
    npx eslint index.js

to fix problems
    npx eslint . --fix
